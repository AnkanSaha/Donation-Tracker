## Run Locally

Clone the project

```bash
  git clone https://github.com/AnkanSaha/Donation-Tracker.git
```

Go to the project directory

```bash
  cd Donation-Tracker
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
